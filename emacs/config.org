* Customize the start of emacs
  ** Standard start of emacs

  #+begin_src emacs-lisp
  ;; Remove the startup message
  (setq inhibit-startup-message t)

  ;; Remove the menu and other elements
  (scroll-bar-mode -1)
  (tool-bar-mode -1)
  (tooltip-mode -1)
  (set-fringe-mode 10)
  (menu-bar-mode -1)

  (setq visible-bell t)
  #+end_src

  #+begin_src emacs-lisp
  ;; Make ESC do what it should
